# vc_manager_bot
## what is this repo?
this is a bot to manage voice channels made with [D++](https://dpp.dev/), an Discord API wrapper.

## how to use?
1. create `token.txt` and paste token on it.
1. install [D++](https://dpp.dev/installing.html).
1. compile yourself: `cmake . && make`.
1. just execute binary.
